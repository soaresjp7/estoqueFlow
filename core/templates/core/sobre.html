Sobre o EstoqueFlow

O EstoqueFlow é um sistema de administração de estoques desenvolvido para facilitar o controle de produtos de pequenos e médios negócios.

Nossa Missão

Oferecer uma solução simples, rápida e eficiente para que qualquer negócio possa controlar seu estoque sem complicações.

Tecnologias Utilizadas

  • Python + Django
  • Bootstrap 5
  • SQLite / PostgreSQL

Desenvolvido por

Projeto acadêmico desenvolvido para a disciplina de Desenvolvimento Web.
